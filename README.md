Bu sayfa, HTML ve JavaScript kullanılarak hazırlanmış eğlenceli bir "Matrix" temalı animasyon örneğidir.
Ziyaretçi fareyle veya dokunarak sahte bir “Kapat” butonuna geldiğinde belirtilen YouTube bağlantısına yönlendirilir.
Basit, tamamen tarayıcıda çalışan ve herhangi bir alan adı (domain) satın almadan GitHub Pages üzerinde barındırılabilir.

Kullanım :

Bu repoyu kendi hesabına forkla veya klonla.

index.html dosyasını düzenle:

VIDEO_URL sabitinin içine istediğin YouTube linkini gir.

Dosyayı kaydet, commit et ve GitHub Pages’i Settings → Pages → Source: main/(root) şeklinde etkinleştir.

Site otomatik olarak https://kullaniciadi.github.io/proje-adi adresinden yayına girer.
